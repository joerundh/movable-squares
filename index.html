Movable Squares

  • Press "+" to create a movable element in a random spot;
  • Press "-" to delete the movable element last added;
  • Click and hold on an element to drag it around;
  • Double-click on an element to select it, and use the arrow keys to move it around;
  • Press Ctrl + Arrow key to make a selected element jump to an edge;
  • Several elements can be selected and moved at the same time.
